How To Connect

This is pretty simple once the Sphero is ready (connected and listening), you just need three lines of code and you are set, Gobot and the Sphero driver take care of the rest.

Install the package with: go get -d -u gobot.io/x/gobot/... && go install github.com/hybridgroup/gobot/platforms/sphero
